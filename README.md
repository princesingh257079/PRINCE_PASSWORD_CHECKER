# 🔐 Password Strength Checker (Python)

## 📌 Description
This project checks the strength of a password based on:
- Length
- Uppercase & lowercase letters
- Numbers
- Special characters

## 🚀 Features
- Detects Weak, Medium, Strong passwords
- Gives suggestions to improve password
- Simple and user-friendly

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open the file in VS Code / Jupyter Notebook
2. Run the script
3. Enter a password

## 📊 Output Example
Weak ❌  
Medium ⚠️  
Strong ✅
